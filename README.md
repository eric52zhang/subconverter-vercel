# subconverter-vercel
subconverter-vercel 订阅转换，Vercel可部署

部署提示：

将这个项目上传到 GitHub；

登录 Vercel，导入仓库；

在设置中选择：

Framework：Other

Build Command：npm run build

Output Directory：dist

部署即可！

可以先在本地运行测试:

安装依赖: npm install

构建项目：npm run build

启动开发服务器：npm run serve

备注：基于cm大佬项目https://github.com/cmliu/SUBWEB ，我改进了vercel支持
